# 🛒 Full Stack E-Commerce Website

A complete E-Commerce web application built using:

- Node.js
- Express.js
- MySQL
- HTML, CSS, JavaScript
- JWT Authentication

## ✨ Features

👤 User Login & Register  
🛒 Add to Cart  
➕ Quantity Management  
⭐ Reviews & Ratings  
📦 Stock System  
🔍 Search Bar  
📞 Contact Form  
🛠️ Admin Panel  

## 🚀 Run Locally

npm install  
node server.js  

Open → http://localhost:5000
