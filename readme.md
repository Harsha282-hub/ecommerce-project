# 🛒 Full Stack E-Commerce Web Application

🚀 A production-ready E-Commerce platform with secure authentication, cart system, admin dashboard, and modern UI.

🔗 Live Demo: http://localhost:5000  
💻 GitHub: https://github.com/Harsha282-hub/ecommerce-project

---

## 📌 Features

### 👤 User
- Login / Register (JWT)
- Browse products
- Search products
- Add to cart
- Increase / decrease quantity
- Remove from cart
- Stock indicator
- Contact form

### 🛠️ Admin
- Add products
- Manage stock
- Delete products

---

## 🧱 Tech Stack

Frontend: HTML, CSS, JavaScript  
Backend: Node.js, Express.js  
Database: MySQL  
Auth: JWT  

---

## ⚙️ Run Locally

npm install  
node server.js  

Open → http://localhost:5000
